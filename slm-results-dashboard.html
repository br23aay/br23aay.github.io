PRIVACY-FIRST ARCHITECTURE
Every component runs on the local machine — Phi-3 Mini via Ollama for generation, ChromaDB for vector retrieval, SQLite for query logging. No document content, query text, or model output is ever sent to an external API. Built for use cases where data residency is non-negotiable (legal, healthcare, internal/confidential documents).
